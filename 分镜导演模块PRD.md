# 分镜导演模块 PRD

## 1. 页面整体布局

分镜导演模块采用暗色主题（主色调：桃红色 `#ff6b9d`），页面由以下区域组成：

- 左侧导航栏（固定宽度 200px）
- 主内容区（自适应宽度），内部分为：
  - 顶部工具栏（含一键导出按钮）
  - 中间内容区（三栏布局）：左侧分镜操作面板 / 中间视频预览区 / 右侧素材资产库
  - 底部区域：左侧生成操作区 + 右侧时间轴/故事板面板

---

## 2. 左侧导航栏

垂直排列的功能入口列表，包含以下模块：

| 序号 | 名称 | 说明 |
|------|------|------|
| 1 | 工作台 | 项目总览 |
| 2 | 剧本工坊 | 剧本编辑与推理 |
| 3 | 选角中心 | 角色选定 |
| 4 | 分镜导演 | 当前模块（高亮状态） |
| 5 | 道具管理 | 道具素材管理 |
| 6 | 生成队列 | 生成任务队列 |
| 7 | 后期精修 | 后期处理 |
| 8 | 成片输出 | 最终输出 |

交互规则：
- 点击导航项切换高亮状态，当前激活项显示主色背景 + 发光阴影
- 导航栏固定不滚动

---

## 3. 顶部工具栏

位于主内容区最上方，右对齐布局。

### 3.1 一键导出按钮
- 样式：主色背景，带发光阴影
- 功能：将当前项目所有分镜的视频内容导出为视频剪辑工程文件（如 XML/EDL 等剪辑软件可识别的工程格式）
- 点击后显示顶部 Toast 提示"正在导出..."

---

## 4. 左侧分镜操作面板

### 4.1 选集与批量生成

面板顶部为一行横向排列的两个控件：

- 选集下拉框：选择当前操作的集数（如"第1集：初次相遇"）
- 批量生成按钮：主色（桃红色）背景，带发光阴影，点击弹出批量生成设置面板（详见 8.3）

### 4.2 分镜标题

显示当前选中分镜的编号，如"分镜1"。随分镜切换自动更新。

### 4.3 分镜原文

- 显示当前分镜的剧本原文内容
- 右下角有编辑按钮（hover 时显示），点击后原文区域变为可编辑的 textarea
- 编辑完成方式：失焦自动保存 / Ctrl+Enter 保存
- 保存后同步更新到分镜数据中
- 分镜原文区域固定不滚动

### 4.4 历史生成卡片区

- 位于分镜原文下方，展示当前分镜的所有历史生成结果
- 卡片高度 140px，宽度 100%，9:16 竖版图片居中显示
- 最新生成的结果插入到最上方（prepend）
- 仅此区域可滚动查看（overflow-y: auto），标题和原文固定不动
- hover 卡片时从底部滑出覆盖层，显示：
  - 生成时使用的提示词文本（最多4行截断）
  - 两个操作按钮：引用、下载

#### 引用按钮
- 将该历史素材添加到参考图堆叠区
- 素材类型保持原始类型（视频类型的历史素材引用为 video，图片类型引用为 image）
- 同时将该历史素材的提示词填入提示词输入框

#### 下载按钮
- 触发浏览器下载该素材图片

---

## 5. 中间视频预览区

### 5.1 预览操作按钮

预览框上方有两个并排按钮：

- 字幕擦除：对当前视频进行字幕擦除处理
- 视频剪裁：打开视频剪裁弹窗（详见 9.1）

显示规则：
- 仅当前预览内容为视频类型时显示
- 图片类型内容时隐藏

### 5.2 预览框

- 最大宽度 420px，1:1 正方形比例
- 内部图片以 9:16 竖版比例居中显示
- 空分镜（无历史生成）显示占位图标
- 布局稳定：添加/删除分镜不影响预览框位置，仅更新内容

### 5.3 预览历史小图框

- 位于预览框右侧，纵向排列
- 每个小图框 60x60px，带圆角和边框
- 左下角显示类型图标：视频类型显示播放三角图标，图片类型显示图片图标
- 默认选中最上方（最新）的小图框
- 最新生成的结果插入到最上方（prepend）
- 点击小图框：
  - 更新预览框显示对应素材
  - 更新预览操作按钮的显示/隐藏（视频显示，图片隐藏）
  - 如果选中的是图片类型，当前分镜时长自动设为 2 秒并重建时间轴

---

## 6. 右侧素材资产库

### 6.1 标签页切换

顶部四个标签页：角色、场景、道具、历史。点击切换对应内容面板。

### 6.2 角色 / 场景标签页

- 顶部有"新增角色"/"新增场景"按钮
- 素材以合集形式展示，每个合集显示封面图 + 名称 + 数量
- 合集为 2 列网格布局
- 点击合集封面展开：
  - 隐藏封面，在原位显示展开网格
  - 顶部有返回按钮（带合集名称），点击收起
  - 展开网格内每个素材 hover 显示"引用"和"编辑"按钮
  - 同一时间只能展开一个合集
- 引用按钮：将素材作为图片类型添加到参考图堆叠区

### 6.3 道具标签页

- 顶部有"新增道具"按钮
- 素材以 2 列网格直接展示
- 每个素材 hover 显示"引用"和"编辑"按钮
- 引用按钮：将素材作为图片类型添加到参考图堆叠区

### 6.4 历史标签页

- 顶部有两个筛选下拉框：类型筛选（全部/视频/图片）、选集筛选
- 素材以 2 列网格展示
- 每个素材 hover 显示"引用"按钮
- 引用按钮：将素材作为图片类型添加到参考图堆叠区

### 6.5 素材类型规则

| 素材来源 | 引用类型 | 说明 |
|----------|----------|------|
| 历史生成卡片（引用按钮） | 保持原始类型（video/image） | 视频生成的为 video，图片生成的为 image |
| 角色/场景/道具素材 | image | 静态素材统一为图片类型 |
| 历史标签页素材 | image | 作为参考图引用 |
| 文件上传 | 根据文件类型自动判断 | video/* 为 video，image/* 为 image |

## 7. 生成操作区

位于底部左侧，与左侧分镜操作面板上下连通（共享背景和边框）。

### 7.1 参考图堆叠区

- 位于生成操作区顶部，向上溢出显示
- 默认显示一个虚线框的添加按钮（支持点击上传和拖拽上传）
- 添加参考图后：
  - 图片以堆叠方式显示（不同旋转角度叠放在同一位置）
  - 添加按钮缩小为圆形，移至堆叠右下角
  - hover 堆叠区时展开为横向排列，每张图显示删除按钮
- 视频类型素材在左下角显示播放三角图标标记
- 图片类型素材无额外标记
- 点击参考图的交互：
  - 视频类型：弹出视频素材选取弹窗（详见 9.2）
  - 图片类型：无操作
- 生成完成后自动清空所有参考图

### 7.2 提示词优化按钮

- 位于生成操作区右上角，魔法棒图标
- 点击后提示词输入框显示"正在优化提示词..."，1 秒后在原文末尾追加优化后缀（电影级调色、IMAX画质等）

### 7.3 提示词输入框

- contenteditable 的 div 元素，支持富文本输入
- 空状态显示 placeholder：
  - 无参考图时："请输入提示词..."
  - 有参考图时："使用@快速调用参考内容"
- 支持 @ 提及功能：
  - 输入 @ 字符时，在输入框上方弹出下拉菜单
  - 下拉菜单显示所有已添加的参考图（24x24 缩略图 + 名称）
  - 支持模糊搜索过滤
  - 选择后在输入框中插入带样式的 @标签（主色边框和文字，不可编辑）
- 生成完成后自动清空提示词内容

### 7.4 生成控制栏

横向排列的控件组：

| 控件 | 说明 |
|------|------|
| 生成类型下拉框 | "视频生成" / "图片生成"，切换后旁白音色按钮隐藏/显示 |
| 旁白音色按钮 | 仅视频生成模式显示，点击弹出音色选择弹窗（详见 8.1） |
| 生成设置按钮 | 点击弹出生成设置面板（详见 8.2） |
| 生成按钮 | 主色背景，左侧显示积分消耗数字（如"25"），点击触发生成 |

### 7.5 生成流程

1. 点击生成按钮后：
   - 按钮显示旋转加载动画 + "生成中..."文字，按钮禁用
   - 3 秒后显示勾选图标 + "成功"，背景变绿
   - 顶部 Toast 提示"视频生成成功！"或"图片生成成功！"
2. 生成完成后：
   - 新结果添加到当前分镜的历史数据（unshift 到最前）
   - 左侧历史卡片区顶部插入新卡片
   - 预览历史小图框顶部插入新图框并自动选中
   - 更新预览框显示最新结果
   - 更新时间轴分镜卡片缩略图
   - 清空提示词输入框
   - 清空参考图堆叠区
   - 2 秒后按钮恢复原始状态

## 8. 弹窗面板

### 8.1 旁白音色弹窗

- 标题："旁白音色"
- 筛选区：性别（全部/男/女）、年龄（全部/幼儿/少年/青年/中年/老年）
- 音色网格：4 列布局，每项显示音量图标 + 音色名称
- 选中项高亮为主色背景
- 确认按钮：保存选择并更新生成控制栏的音色按钮文字

### 8.2 生成设置弹窗（单个分镜）

顶部为品牌标签页切换，内容根据当前生成类型（视频/图片）动态渲染。

#### 视频生成模式

品牌选项：

| 品牌 | 模型版本 | 时长选项 | 清晰度 | 比例 |
|------|----------|----------|--------|------|
| 即梦 | Seedance 2.0, Seedance 1.5pro | 自动匹配, 5s, 10s, 15s | 720p（2.0）/ 720p, 1080p（1.5pro） | 16:9, 9:16, 4:3, 3:4, 1:1 |
| 万相 | Wan 2.6, Wan 2.5, Wan 2.0 | 5s, 8s, 10s | 720p, 1080p | 16:9, 9:16, 4:3, 3:4, 1:1 |
| Vidu | Vidu 2.0, Vidu 1.5, Vidu 1.0 | 4s, 8s, 16s | 720p, 1080p | 16:9, 9:16, 4:3, 3:4, 1:1 |

附加选项：开启音效开关（同时生成音效和视频）

特殊规则：
- 即梦品牌的清晰度选项与模型版本关联：Seedance 2.0 仅支持 720p，Seedance 1.5pro 支持 720p 和 1080p
- 切换模型版本时，如当前清晰度不在新模型支持范围内，自动重置为第一个可用选项

#### 图片生成模式

品牌选项：

| 品牌 | 模型版本 | 比例 |
|------|----------|------|
| 即梦 | Seedream 2.0, Seedream 1.5 | 16:9, 9:16, 4:3, 3:4, 1:1 |
| 万相 | Wan-img 2.0, Wan-img 1.5 | 16:9, 9:16, 4:3, 3:4, 1:1 |
| Vidu | Vidu-img 2.0, Vidu-img 1.0 | 16:9, 9:16, 4:3, 3:4, 1:1 |

图片模式无清晰度选项、无时长选项、无音效开关。

#### 通用交互
- 切换品牌标签页时，模型版本、比例等重置为该品牌的第一个选项
- 确认按钮保存设置并显示 Toast 提示

### 8.3 批量生成设置弹窗

- 仅显示视频生成设置（无图片模式切换）
- 内容与视频生成模式的生成设置面板完全一致（品牌标签页、模型版本、时长、清晰度、比例、音效开关）
- 确认按钮左侧显示积分消耗数字，计算方式：分镜总数 × 25
- 确认后显示 Toast 提示批量生成已提交，包含分镜数量和设置摘要

## 9. 视频剪裁与素材选取弹窗

### 9.1 视频剪裁弹窗

- 入口：预览操作区的"视频剪裁"按钮
- 顶部标题："视频剪裁"
- 预览区：16:9 比例的黑色背景区域，居中显示当前预览图
- 时间轴区域：
  - 时间信息显示：当前帧数（如"45帧"）/ 选中范围秒数（如"3.50秒"）
  - 播放按钮（迷你版）：播放/暂停当前选中范围
  - 帧条：由缩略图组成的帧序列
  - 范围选择器：可拖拽的起止手柄 + 播放头
- 底部操作：
  - 重置修改：恢复默认状态（起始0%，结束100%，播放头归零）
  - 应用按钮：保存剪裁结果，更新分镜时长并重建时间轴
- 时间范围基于当前分镜的 duration 字段，非固定值

### 9.2 视频素材选取弹窗

- 入口：点击参考图堆叠区中的视频类型素材
- 顶部标题："视频素材选取"
- 模式切换：选帧 / 选片段（两个按钮切换，当前模式高亮为主色）
- 预览区和时间轴区域与视频剪裁弹窗相同
- 底部操作：
  - 重置修改
  - 应用按钮：
    - 选帧模式：将当前帧作为图片素材添加到参考图，命名格式"[来源名]-[帧数]帧"
    - 选片段模式：将选中片段作为素材添加到参考图，命名格式"[来源名]-[起始时间]-[结束时间]"

### 9.3 时间轴拖拽交互

- 起始手柄：拖拽调整选中范围起点，不可超过终点
- 结束手柄：拖拽调整选中范围终点，不可低于起点
- 播放头：拖拽在选中范围内移动
- 范围区域：整体拖拽移动选中范围
- 播放功能：从当前播放头位置播放到范围终点，到达终点自动停止

## 10. 时间轴面板

### 10.1 时间轴头部

- 播放按钮：圆形主色按钮，切换播放/暂停状态
- 时间显示：当前时间 / 总时长（格式 MM:SS）
- 切换视图按钮：在时间轴视图和故事板视图之间切换

### 10.2 时间轴视图

- 分镜卡片横向排列，统一宽度 100px
- 每个卡片显示：
  - 缩略图（16:9 比例，9:16 竖版图片居中）
  - 默认显示分镜编号标签（hover 时隐藏）
  - hover 时显示底部信息栏：时长 + 复制按钮 + 删除按钮
- 分镜之间、第一个分镜之前、最后一个分镜之后均有添加分镜触发器（hover 时显示圆形加号按钮）
- 空分镜显示占位图标

### 10.3 播放头

- 红色竖线 + 顶部圆点，可拖拽定位
- 播放时自动移动，速度根据每个分镜的 duration 字段变化（卡片宽度统一，播放头在不同分镜上移动速度不同）
- 拖拽或点击时间轴空白区域可定位播放头
- 播放头位置变化时自动同步：
  - 更新左侧分镜操作面板（标题、原文、历史卡片）
  - 更新中间预览区
  - 高亮对应的时间轴分镜卡片
- 播放到末尾自动停止并归零

### 10.4 分镜数据结构

每个分镜包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| id | number | 分镜编号，自动递增 |
| img | string | 缩略图 URL |
| text | string | 分镜原文 |
| duration | number | 时长（秒），新建分镜默认 2 秒 |
| history | array | 历史生成记录数组 |

history 数组中每项：

| 字段 | 类型 | 说明 |
|------|------|------|
| src | string | 素材 URL |
| type | string | 'video' 或 'image' |
| prompt | string | 生成时使用的提示词 |

### 10.5 分镜操作

| 操作 | 说明 |
|------|------|
| 点击分镜卡片 | 选中该分镜，更新左侧面板和预览区 |
| 添加分镜 | 在指定位置插入新空分镜（duration=2s），自动选中新分镜 |
| 删除分镜 | 确认后删除，至少保留 1 个分镜（否则 Toast 提示） |
| 复制分镜 | 复制当前分镜数据 |

## 11. 故事板视图

### 11.1 布局变化

切换到故事板视图时：
- 中间预览区和右侧素材库隐藏
- 左侧分镜操作面板保留，生成操作区移入其中
- 故事板网格占据右侧全部空间
- 时间轴头部（播放按钮、时间、切换按钮）移至底部

### 11.2 故事板网格

- 4 列网格布局，每个卡片 16:9 比例
- 卡片内容：缩略图（9:16 竖版居中）+ 分镜编号标签
- hover 时显示覆盖层：
  - 分镜原文（最多 3 行截断）
  - 编辑按钮：点击后原文变为可编辑 textarea，失焦或 Ctrl+Enter 保存
  - 删除按钮：确认后删除分镜
- 点击卡片选中该分镜（边框变为主色），更新左侧面板

### 11.3 添加分镜触发器

以绝对定位的圆形加号按钮形式出现在：
- 第一个卡片左侧
- 相邻卡片之间（同行：中间位置；跨行：当前行末尾 + 下一行开头）
- 最后一个卡片右侧

hover 时显示，点击后在对应位置插入新空分镜。

## 12. 数据流转与业务规则

### 12.1 剧本推理 → 分镜导演

当剧本推理（剧本工坊模块）完成后：
- 系统自动为每一集的每一个分镜生成初始数据，包括：
  - 分镜原文（text）：来自剧本推理结果
  - 提示词（prompt）：来自剧本推理生成的描述
  - 参考图数据：来自剧本推理关联的素材
- 此时分镜尚未生成图片或视频，history 数组为空
- 用户进入分镜导演模块后可以：
  - 浏览和编辑每个分镜的原文
  - 浏览和编辑每个分镜的提示词
  - 调整参考图
  - 逐个点击"生成"按钮生成单个分镜的视频/图片
  - 点击"批量生成"按钮一次性生成当前集所有分镜的视频

### 12.2 批量生成流程

1. 用户点击"批量生成"按钮
2. 弹出批量生成设置面板（仅视频模式）
3. 用户选择品牌、模型、时长、清晰度、比例等参数
4. 确认后系统按设置参数批量生成当前集所有分镜的视频
5. 生成完成后每个分镜的 history 中会有对应的生成结果
6. 页面呈现为已生成状态（即 demo 当前展示的样子）

### 12.3 一键导出

用户制作完所有分镜内容后，点击"一键导出"按钮：
- 系统将所有分镜的视频素材按时间轴顺序组装
- 导出为视频剪辑软件可识别的工程文件格式
- 用户可在专业剪辑软件中进一步编辑

## 13. Toast 提示规则

- 位置：页面顶部居中，从上方滑入
- 动画：滑入显示 → 停留 2 秒 → 滑出消失
- 样式：黑色半透明背景，主色边框和发光阴影
- 仅在以下重要操作时显示：

| 场景 | 提示内容 |
|------|----------|
| 生成成功 | "视频生成成功！" / "图片生成成功！" |
| 导出 | "正在导出..." |
| 生成设置保存 | 设置摘要信息 |
| 批量生成提交 | 分镜数量 + 设置摘要 |
| 删除最后一个分镜 | "至少保留一个分镜" |

以下操作不显示 Toast：导航切换、参考图增删、原文编辑保存、播放/暂停、视图切换、添加分镜、音色选择、素材引用、剪裁操作等。
