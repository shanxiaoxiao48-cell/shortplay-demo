# 分镜导演模块

暗色主题的分镜导演交互界面，用于视频/图片生成、分镜管理和时间轴编辑。

## 功能特性

- 分镜操作面板：编辑分镜原文、查看历史生成记录
- 视频预览区：实时预览、视频剪裁、字幕擦除
- 素材资产库：角色、场景、道具、历史素材管理
- 生成控制：参考图堆叠、提示词输入、@ 提及功能
- 时间轴/故事板：双视图切换、播放控制、分镜编辑
- 批量生成：一键批量生成整集视频

## 技术栈

- 纯 HTML + CSS + JavaScript
- 暗色主题 + 桃红色主调
- 响应式布局

## 本地运行

直接打开 `index.html` 或使用本地服务器：

```bash
# Python
python -m http.server 8000

# Node.js
npx serve
```
